How to use icon:
	Flaticon.get('<icon-name>')

Example:
	text: Flaticon.get('morning')

View all icon name in flaticon.css file:
	- From repo: \mobile-base\flaticon
	- From Qt Creator: \home-icons
	
Download all icon:
    - Go to https://www.flaticon.com/collections/Mzc4MTM2NA==
    - Add the icon package to your collection and download as iconfont format
